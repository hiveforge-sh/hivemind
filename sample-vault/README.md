# Sample Worldbuilding Vault

This is a sample Obsidian vault demonstrating the Hivemind MCP server structure.

## Structure

- `Characters/` - Character notes with detailed profiles
- `Locations/` - Location notes with geography and inhabitants
- `Events/` - Historical events and timeline entries
- `Factions/` - Organizations, houses, and groups
- `Assets/` - AI-generated images and their metadata

## Example Characters

- [[Eddard Stark]] - Lord of Winterfell, Warden of the North

## Example Locations

- [[Winterfell]] - Ancestral seat of House Stark

## Usage

This vault can be queried via the Hivemind MCP server using tools like:
- `query_character` - Get detailed character information
- `query_location` - Get location details
- `search_vault` - Search across all content
